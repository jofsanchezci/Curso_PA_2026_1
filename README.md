## Condicionales:
Las condicionales en Python permiten ejecutar código basado en si una condición es verdadera o falsa. Son fundamentales para el control de flujo en programación.

Sintaxis básica
El if evalúa una condición y ejecuta el bloque indentado si es True.

La estructura es:
```python
if condicion:
    # código a ejecutar
```
